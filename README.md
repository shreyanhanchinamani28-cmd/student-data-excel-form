# Student Data Form using Python and Excel Integration

A Python GUI-based Student Data Entry System using Tkinter, Pandas, and Excel integration.

## Features

- Simple GUI form using Tkinter
- Student data entry system
- Stores data into Excel automatically
- Existing and new data saved in same Excel file
- Automatic Excel file opening after saving
- Password hidden using '*' symbol
- Input validation for empty fields

## Technologies Used

- Python
- Tkinter
- Pandas
- OpenPyXL
- Excel Integration

## How It Works

1. Run the Python program.
2. A pink GUI form window opens.
3. Enter student details:
   - USN
   - Semester
   - Branch
   - College
   - Username
   - Password
4. Click on "SAVE TO EXCEL".
5. Data gets stored into `students.xlsx`.
6. Existing and new records are merged together.
7. Excel file opens automatically after saving.

## Output

- GUI Student Form
- Automatic Excel File Creation
- Student Data Storage System
- Excel Automation
